**פונקציה קמורה:** $f$ נקראת **קמורה** בקטע אם לכל $x_1, x_2$ ו-$\lambda \in [0, 1]$: $f(\lambda x_1 + (1-\lambda) x_2) \leq \lambda f(x_1) + (1-\lambda) f(x_2)$. שווה ערך לקעירות כלפי מעלה.
