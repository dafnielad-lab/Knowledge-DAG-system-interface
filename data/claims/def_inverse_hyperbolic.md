**פונקציות היפרבוליות הפוכות:** $\text{arsinh}\, x = \ln(x + \sqrt{x^2 + 1})$, $\text{arcosh}\, x = \ln(x + \sqrt{x^2 - 1})$ (עבור $x \geq 1$), $\text{artanh}\, x = \tfrac{1}{2} \ln\!\frac{1+x}{1-x}$ ($|x| < 1$).
