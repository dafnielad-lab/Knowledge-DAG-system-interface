**אי-שוויון מורכב:** אי-שוויון מורכב הוא שילוב של שני אי-שוויונים פשוטים במשתנה $x$ שאחת משתי קישוריות לוגיות מקשרת ביניהם:

1. **אי-שוויון מורכב מסוג "וגם" (AND, חיתוך):** $a < x < b$ — הצורה הקצרה ל-$(a < x) \land (x < b)$, כלומר $x$ מקיים את שני האי-שוויונים בו-זמנית. קבוצת הפתרון היא $\{x \in \mathbb{R} : a < x \text{ וגם } x < b\}$, וכאשר $a < b$ זהו הקטע הפתוח $(a, b)$.

2. **אי-שוויון מורכב מסוג "או" (OR, איחוד):** $x < a$ או $x > b$ — צירוף לוגי $(x < a) \lor (x > b)$. קבוצת הפתרון היא $\{x \in \mathbb{R} : x < a\} \cup \{x \in \mathbb{R} : x > b\}$.

כל אחד מהאי-שוויונים המרכיבים יכול להיות חזק ($<$, $>$) או חלש ($\leq$, $\geq$).
